# Session Closure Protocol

**Статус:** Canonical  
**Версия:** v_1_0

## Назначение

Настоящий протокол определяет **минимально обязательные условия корректного закрытия сессии** (чата) между пользователем и AI-ассистентом.

Протокол фиксирует:
- когда сессия считается корректно завершённой;
- какие артефакты **обязаны** быть подготовлены;
- в каких случаях закрытие сессии запрещено.

Операционные инструкции по выполнению требований настоящего протокола
содержатся в `session_finalization_playbook.md`.

---

## 1. Область применения

Протокол применяется:
- при завершении **любой рабочей сессии**;
- при переходе в новый чат в рамках одного проекта;
- при остановке работы из-за риска потери контекста.

Протокол обязателен к применению вне зависимости от длительности сессии.

---

## 2. Обязательность закрытия сессии

Закрытие сессии **обязательно**, если выполнено хотя бы одно из условий:

1. В сессии были приняты архитектурные, проектные или процессные решения.
2. В сессии были выполнены изменения:
   - в коде;
   - в конфигурации;
   - в документации;
   - в инфраструктуре.
3. Сессия завершает логический этап работы.
4. Пользователь явно инициировал закрытие сессии.
5. Существует риск потери контекста при продолжении работы.

---

## 3. Обязательные артефакты закрытия

Корректное закрытие сессии **невозможно**, если не подготовлены следующие артефакты:

1. `session_closure_log_YYYY_MM_DD_vX.Y.md`
2. При необходимости обновления:
   - `project_summary_vX.Y.md`
   - `project_history_log.md`
   - `techspec_vX.Y.md`
   - `plan_vX.Y.md`

Отсутствие обязательных артефактов означает, что сессия **не считается закрытой**.

---

## 4. Запреты

Запрещено закрывать сессию:

1. Без фиксации принятых решений.
2. Без описания фактически выполненных действий.
3. С устными или неявными договорённостями.
4. С противоречиями между артефактами.
5. С «тихими» правками без версии.

---

## 5. Ответственность сторон

### Пользователь обязан:
- инициировать закрытие сессии;
- подтвердить корректность подготовленных артефактов.

### AI-ассистент обязан:
- выявить необходимость закрытия сессии;
- подготовить обязательные артефакты;
- указать на невозможность закрытия при нарушении протокола.
