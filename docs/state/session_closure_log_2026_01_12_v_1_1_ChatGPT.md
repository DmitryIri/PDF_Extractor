## 10. Приложение A — Safe cleanup (quarantine pattern)

> Выполнять только при реальной необходимости cleanup `/srv/pdf-extractor`.

```bash
cd /srv/pdf-extractor
TS="$(date +%Y%m%d_%H%M%S)"
mkdir -p "_quarantine/${TS}"

# пример: вместо удаления — перемещение
[ -d venv ] && mv venv "_quarantine/${TS}/venv"
[ -d logs ] && mv logs "_quarantine/${TS}/logs"

mkdir -p logs

# проверки
ls -la
find _quarantine -maxdepth 2 -type d -print
```
